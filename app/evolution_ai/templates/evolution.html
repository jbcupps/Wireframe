How It Works

The evolutionary algorithm uses principles inspired by natural selection to find optimal Sub-SKB configurations:

  1. Initialization: Creates a population of random Sub-SKB configurations
  2. Evaluation: Calculates fitness for each individual based on how well they match target criteria
  3. Selection: Chooses individuals with higher fitness to become parents
  4. Crossover: Creates new individuals by combining parameters from parents
  5. Mutation: Randomly modifies some parameters to maintain diversity
  6. Iteration: Repeats the process for multiple generations

The fitness function determines how well a Sub-SKB configuration matches the desired properties:

  • Twist Sum: The sum of the absolute twist values along all axes (|X| + |Y| + |Z|)
  • Loop Factor: Influences the number of loops in the manifold

Fitness is calculated based on the proximity to target values, weighted by their importance:

\begin{align} F_{twist} &= \frac{1}{1 + |T_{actual} - T_{target}|} \\ F_{loop} &= \frac{1}{1 + |L_{actual} - L_{target}|} \\ Fitness &= w_{twist} \cdot F_{twist} + w_{loop} \cdot F_{loop} \end{align}

The results of the evolutionary algorithm can be interpreted in several ways:

  • Fitness Graph: Shows how the population fitness improves over generations
  • Best Individual: The Sub-SKB configuration with the highest fitness
  • Population Grid: Visual representation of all individuals in the final population

For the best individual, we predict particle properties using a machine learning model trained on hypothetical correlations between topological features and physical properties.
